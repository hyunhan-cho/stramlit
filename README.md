# 설문 데이터 분석 대시보드

주피터 노트북(`Untitled12.ipynb`)에서 수행한 설문 데이터 분석을 인터랙티브한 웹 대시보드로 구현한 프로젝트입니다.

## 📊 주요 기능

- **연령/성별/학력별 요인 분석**: 그룹을 선택하여 6개 주요 요인(SNS 이용량, 충동구매 성향, 사회적 비교, 쇼핑 후회/태도, 광고 인식/신뢰, 구매 의도)의 평균과 분포를 시각화
- **ANOVA 통계 분석**: 선택한 그룹 간 유의미한 차이가 있는지 자동 검증 (P-value 계산)
- **상관관계 히트맵**: 요인 간의 상관관계를 한눈에 파악
- **인터랙티브 차트**: Plotly 기반 동적 그래프로 상세 데이터 확인 가능

## 🚀 실행 방법

### 1. 필요한 라이브러리 설치

```bash
pip install pandas numpy scipy plotly streamlit
```

### 2. 대시보드 실행

```bash
streamlit run streamlit_app.py
```

실행 후 브라우저가 자동으로 열리며 `http://localhost:8501`에서 대시보드를 확인할 수 있습니다.

## 📁 파일 구조

- `streamlit_app.py` - Streamlit 기반 웹 대시보드 메인 파일
- `Untitled12.ipynb` - 원본 주피터 노트북 분석 파일 (참고용)
- `exp - 설문지 응답 시트1 (1).csv` - 설문 응답 데이터

## 📈 분석 내용

### 데이터 전처리
- 나이 숫자 변환 및 연령대 그룹화
- SNS 이용량 점수 변환
- 성별/학력 컬럼 자동 탐색

### 요인 점수 계산
6개 주요 요인별로 관련 문항들의 평균 점수를 계산합니다:
1. SNS 이용량
2. 충동구매 성향
3. 사회적 비교
4. 쇼핑 후회/태도
5. 광고 인식/신뢰
6. 구매 의도

### 통계 분석
- 그룹별 요인 평균 비교 (바 차트)
- 단일 요인 분포 분석 (박스플롯)
- ANOVA를 통한 그룹 간 차이 검증
- 요인 간 상관계수 분석 (히트맵)

## 📝 주요 발견사항

- **광고 인식/신뢰**와 **구매 의도** 간에 매우 강한 양의 상관관계 (0.85)
- 연령과 성별은 대부분의 요인에서 통계적으로 유의미한 차이를 보이지 않음
- 학력에 따라 일부 요인(사회적 비교, 쇼핑 후회/태도, 구매 의도)에서 유의미한 차이 발견

## 💡 사용 팁

1. 사이드바에서 **그룹 기준**을 선택하세요 (연령 2그룹/3그룹, 성별, 학력)
2. **요인**을 선택하세요 (전체 요인 또는 개별 요인)
3. 차트에 마우스를 올리면 상세 수치를 확인할 수 있습니다
4. 단일 요인 선택 시 ANOVA 결과를 자동으로 계산하여 표시합니다
